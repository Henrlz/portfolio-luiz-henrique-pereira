## Publicação no GitHub Pages

1. Crie um repositório público no GitHub com um nome que contenha seu nome, por exemplo: `portfolio-luiz-henrique-pereira`.
2. Envie todos os arquivos e pastas deste projeto para o repositório.
3. Acesse Settings > Pages.
4. Em Source, selecione Deploy from a branch.
5. Em Branch, selecione `main` e `/root`.
6. Salve e aguarde o GitHub gerar o link publicado.
